# Jogo do PACMAN em C usando raylib

Uma implementação do jogo do **Pacman** em **C**, utilizando a biblioteca gráfica **raylib**

## Objetivo

O objetivo deste projeto é a recriação do jogo PACMAN utilizando linguagem C e a biblioteca gráfica raylib

## Tecnologias Utilizadas

- Linguagem: **C**
- Biblioteca: **raylib**

## Conceitos aplicados

- Lógica de programação
- Manipulação de arquivos (.txt e .bin)
- Renderização gráfica 2D

## Como foi feito

O jogo foi do zero, usando apenas linguagem de programação básica em C e recursos da biblioteca raylib, além disso foram feitos arquivos .txt para salvar mapas, e arquivos .bin para salvar os dados do jogo

## Como usar

Dentro do projeto há um arquivo executável no qual se pode rodar o jogo. Para rodar dentro de uma IDE (Integrated Development Environments) é necessário configurar o ambiente para habilitar o uso da biblioteca raylib
